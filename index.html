NumPy — Numerical Python

The fundamental package for scientific computing — high-performance multidimensional arrays

What is NumPy?

NumPy (Numerical Python) is the backbone of data science. It provides a powerful N-dimensional array object (ndarray) and tools for efficient numerical computation.

  • N-dimensional array object (ndarray)
  • Sophisticated broadcasting functions
  • Linear algebra, Fourier transform & random number tools
  • C/C++ & Fortran code integration

Creating Arrays

Arrays can be created from Python lists, tuples, or using built-in functions:

Python
import numpy as np

# From a list
arr = np.array([1, 2, 3])
print(arr)

# 2D from nested list
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2)

# From a tuple
arr3 = np.array((1, 3, 2))
print(arr3)

ndarray & Its Attributes

The ndarray is the core NumPy object — a homogeneous, multi-dimensional container. Each array has key attributes:

AttributeDescriptionExample
shapeTuple of array dimensions(2, 3)
ndimNumber of dimensions (axes)2
sizeTotal number of elements6
dtypeData type of elementsint64
itemsizeSize (bytes) of each element8
Python
arr = np.array([[1,2,3],[4,5,6]])
print("Shape:", arr.shape)
print("Dim:", arr.ndim)
print("Size:", arr.size)
print("Type:", arr.dtype)
print("Item size:", arr.itemsize)

# 2D Slicing
arr = np.array([[-1, 2, 0, 4],

Array Operations & Data Types

Python
a = np.array([[1,2],[3,4]])
b = np.array([[4,3],[2,1]])

print("Adding 1:", a + 1)
print("Sum:", a.sum())
print(a + b)

# Data types
x = np.array([1, 2])
print(x.dtype)
x = np.array([1.0, 2.0])
print(x.dtype)

Boolean Indexing

Python
a = np.array([10, 40, 80, 50, 100])
print(a[a > 50])  # Filter: [80 100]

b = np.array([[5,5],[4,5],[16,4]])
print(b[b.sum(-1) % 10 == 0])

Pandas — Data Analysis Library

Powerful data manipulation — Series, DataFrames, cleaning, grouping & aggregation

Pandas Series

A Series is a one-dimensional labeled array, like a single column in a spreadsheet. Each value has an associated index.

Python
import pandas as pd
import numpy as np

# From list
ser = pd.Series(['g','e','e','k','s'])
print(ser)

# From dictionary (keys become index)
ser2 = pd.Series({'Geeks':10, 'for':20})

# With custom index
ser3 = pd.Series(np.arange(3,9),
              index=['a','b','c','d','e','f'])

DataFrame & Selection

A DataFrame is a 2D table with rows and columns.

Python
df = pd.DataFrame({
    'Name':['Tom','nick','krish','jack'],
    'Age':[20,21,19,18]
})
print(df)

# .loc[] - label based
# .iloc[] - integer position
print(df.loc[0])          # First row by label
print(df.iloc[1])         # Second row by position
print(df[['Name']])      # Column selection

Handling Missing Data

Python
dict = {'First Score':[100,90,np.nan,95],
        'Second Score':[30,45,56,np.nan],
        'Third Score':[np.nan,40,80,98]}
df = pd.DataFrame(dict)

# Check missing
print(df.isnull())

# Fill with 0
print(df.fillna(0))

# Drop rows with any NaN
print(df.dropna())

Filtering, Sorting & Grouping

Python
df = pd.DataFrame({
    'Name':['RACHEL','MONICA','PHOEBE'],
    'Age':[30,35,37],
    'Salary':[100000,93000,88000],
    'JOB':['DESIGNER','CHEF','MASUS']
})

# Filter with multiple conditions
print(df.loc[(df['Salary'] >= 100000) &
          (df['Age'] < 40) &
          (df['JOB'].str.startswith('D'))])

# Sort and group
print(df.sort_values(by='Age'))
print(df.groupby('JOB')['Salary'].sum())

Object-Oriented Programming in Python

Organizing code using classes and objects that represent real-world entities

What is OOP?

Object-Oriented Programming organizes code using objects and classes to represent real-world entities.

  • Organizes code into classes and objects
  • Supports encapsulation — group data and methods together
  • Enables inheritance for reusability
  • Allows polymorphism for flexible implementations

Classes & Objects

Python
class Dog:
    species = "Canine"  # Class attribute
    def __init__(self, name, age):
        self.name = name    # Instance attr
        self.age = age
    def bark(self):
        print(f"{self.name} says Woof!")

dog1 = Dog("Buddy", 3)
print(dog1.name)
print(dog1.age)
dog1.bark()

class Calculator:
    def add(self, a, b): return a + b
calc = Calculator()
print(calc.add(5, 3))

The __init__ Method & self

Python
class Person:
    def __init__(self, name, age=18):
        self.name = name
        self.age = age

p1 = Person("Emil")
p2 = Person("Tobias", 25)
print(p1.name, p1.age)
print(p2.name, p2.age)

Dynamic Attribute Access

Python
class Employee:
    def __init__(self, name='Harsh', salary=25000):
        self.name = name
        self.salary = salary

emp = Employee()
print(getattr(emp, 'name'))     # Harsh
print(hasattr(emp, 'salary'))   # True
setattr(emp, 'age', 25)
print(emp.age)                     # 25
delattr(emp, 'salary')
print(emp.salary)                  # Error!

Inheritance in Python

Acquiring properties from a parent class — code reuse, hierarchy & extensibility

Why Inheritance?

  • Code reusability — share attributes across classes
  • Real-world modeling — e.g. Animal → Dog
  • Easy maintenance — update parent, fix all children
  • Method overriding — customize behavior

Basic Inheritance

Python
class Animal:
    def __init__(self, name):
        self.name = name
    def info(self):
        print(f"Animal name: {self.name}")

class Dog(Animal):  # Dog inherits Animal
    def sound(self):
        print(f"{self.name} barks")

d = Dog("Buddy")
d.info()    # Inherited method
d.sound()   # Own method

Inheritance Types — Visual Diagram

Three types of inheritance: Single ← Multiple ← Multilevel

3 Types of Inheritance

1. Single Inheritance

Python
class Parent:
    def func1(self): print("Parent")

class Child(Parent):
    def func2(self): print("Child")

obj = Child()
obj.func1()  # Parent
obj.func2()  # Child

2. Multiple Inheritance

Python
class Father:
    fathername = ""

class Mother:
    mothername = ""

class Son(Mother, Father):
    def parents(self):
        print("Father:", self.fathername)
        print("Mother:", self.mothername)

s1 = Son()
s1.fathername = "RAM"
s1.mothername = "SITA"
s1.parents()

3. Multilevel Inheritance

Python
class Grandfather:
    def __init__(self, name):
        self.grandfathername = name

class Father(Grandfather):
    def __init__(self, fname, gname):
        self.fathername = fname
        Grandfather.__init__(self, gname)

class Son(Father):
    def __init__(self, sname, fname, gname):
        self.sonname = sname
        Father.__init__(self, fname, gname)
    def show(self):
        print(f"Grandfather: {self.grandfathername}")
        print(f"Father: {self.fathername}")
        print(f"Son: {self.sonname}")

s = Son("Prince","Rampal","Lal mani")
s.show()
